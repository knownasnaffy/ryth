ryth

NAME

ryth - Scriptable iwd interface

SYNOPSIS

ryth [-h|--help] <subcommands>

DESCRIPTION

Scriptable iwd interface

OPTIONS

-h, --help
Print help

SUBCOMMANDS

ryth-status(1)
Current wifi status (state, signal, powered, etc.)
ryth-list(1)
List discovered networks (cached)
ryth-scan(1)
Trigger a scan and return updated network list
ryth-connect(1)
Connect to a network by SSID
ryth-disconnect(1)
Disconnect from the current network
ryth-autoconnect(1)
Toggle autoconnect for a known network
ryth-forget(1)
Forget a known network
ryth-known(1)
List known networks
ryth-power(1)
Toggle wifi power
ryth-help(1)
Print this message or the help of the given subcommand(s)
